# 機能設計書

## 1. Todoアイテムの管理

### 1.1 Todoの追加
- **機能**: 新しいTodoを追加
- **入力項目**: タイトル（必須）、説明（任意）、期限（任意）
- **バリデーション**: タイトルは1文字以上100文字以内
- **動作**: Enterキーまたは追加ボタンで登録

### 1.2 Todoの編集
- **機能**: 既存Todoの内容を編集
- **操作**: ダブルクリックで編集モードに移行
- **保存**: Enterキーまたはフォーカスアウトで保存

### 1.3 Todoの削除
- **機能**: Todoを削除
- **操作**: 削除ボタンをクリック
- **確認**: 確認ダイアログを表示

### 1.4 完了/未完了の切り替え
- **機能**: チェックボックスで状態を切り替え
- **UI変化**: 完了時は打ち消し線を表示

## 2. フィルタリング機能

### 2.1 表示切り替え
- **全て**: すべてのTodoを表示
- **未完了**: 未完了のTodoのみ表示
- **完了済み**: 完了済みのTodoのみ表示

## 3. データ永続化

### 3.1 保存
- **タイミング**: Todo追加・編集・削除・状態変更時に自動保存
- **保存先**: LocalStorage（または API経由でDB）
