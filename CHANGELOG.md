## [1.1.0] - 2025-11-23

### Added
- **カテゴリ機能**
  - カテゴリの作成・削除機能
    - カテゴリ名（必須、1-50文字）
    - カテゴリ色設定（カラーピッカー対応）
    - プリセットカラーパレット
  - Todoへのカテゴリ割り当て
    - ドロップダウンでカテゴリ選択
    - カテゴリの色をバッジとして表示
  - カテゴリによるフィルタリング
    - 特定カテゴリのTodoのみ表示
    - 未分類Todoのフィルタリング
    - 既存フィルター（全て/未完了/完了済み）と併用可能

- **優先度機能**
  - 優先度設定（高/中/低）
    - デフォルト値: 中
    - 色分け表示（高=赤、中=黄、低=緑）
  - 優先度によるソート機能
    - 高→中→低の順に表示
    - ワンクリックで並び替え

- **データマイグレーション機能**
  - v1.0.0からv1.1.0への自動マイグレーション
  - 既存Todoへのデフォルト値自動設定
    - categoryId: undefined（未分類）
    - priority: 'medium'
  - Local Storageバージョン管理
    - app_versionキーによるバージョン判定
    - 初回起動時のみ実行

- **新規コンポーネント**
  - CategoryManager: カテゴリ管理UI
  - useCategories: カテゴリ管理用カスタムフック

### Changed
- **Todo型の拡張**
  - `categoryId?: string` プロパティ追加
  - `priority: 'high' | 'medium' | 'low'` プロパティ追加

- **Local Storageキーの追加**
  - `categories`: カテゴリデータの保存用
  - `app_version`: バージョン管理用

- **既存コンポーネントの拡張**
  - TodoForm: カテゴリ選択・優先度選択フィールド追加
  - TodoItem: カテゴリバッジ・優先度バッジ表示
  - TodoFilter: カテゴリフィルタ・優先度ソートボタン追加
  - TodoList: カテゴリ情報の受け渡し対応
  - useTodos: カテゴリフィルタ・優先度ソート機能追加

- **ドキュメント**
  - DESIGN_V2.md: v1.1.0の機能詳細設計
  - CHANGELOG.md: Keep a Changelogフォーマット準拠

## [1.0.0] - 2025-01-15

### Added
- **Todoの追加機能**
  - タイトル入力（必須、1-100文字）
  - 説明入力（任意）
  - 期限設定（任意）
  - Enterキーまたは追加ボタンで登録
  - バリデーション実装

- **Todoの編集機能**
  - ダブルクリックで編集モードに移行
  - Enterキーまたはフォーカスアウトで保存
  - 編集中のキャンセル機能

- **Todoの削除機能**
  - 削除ボタンで削除
  - 確認ダイアログ表示

- **完了/未完了の切り替え**
  - チェックボックスで切り替え
  - 完了時に打ち消し線表示

- **フィルタリング機能**
  - 全て表示
  - 未完了のみ表示
  - 完了済みのみ表示

- **データ永続化**
  - Local Storageによる自動保存
  - アプリ起動時の自動読み込み
  - エラーハンドリング実装

- **レスポンシブデザイン**
  - モバイルファーストの設計
  - タブレット・デスクトップ対応

- **技術スタック**
  - Next.js 16.0.3 (App Router)
  - React 19.2.0
  - TypeScript 5
  - Tailwind CSS 4
  - Local Storage（データ永続化）

- **ドキュメント**
  - CLAUDE.md: 開発ガイドライン
  - DESIGN.md: 機能設計書
  - MOCKUP.md: 画面設計・UI仕様
  - README.md: ユーザー向けドキュメント

## バージョン管理方針

このプロジェクトは [Semantic Versioning](https://semver.org/lang/ja/) に従います:

- **MAJOR version (X.0.0)**: 破壊的変更、大規模リニューアル
- **MINOR version (1.X.0)**: 新機能追加（後方互換性あり）
- **PATCH version (1.0.X)**: バグ修正、小規模改善（後方互換性あり）

## 変更の種類

- **追加**: 新機能の追加
- **変更**: 既存機能の変更
- **非推奨**: 今後削除される機能（互換性のため残す）
- **削除**: 削除された機能
- **修正**: バグ修正
- **セキュリティ**: セキュリティ関連の修正
