# 画面モック・UI設計

## モック作成の方針
- 最小限のHTML構造で要素を配置
- CSSは基本的な配置とサイズのみ（枠線表示など）
- 色やフォントなどの装飾は機能実装後に対応

## 画面構成

### メイン画面
```
┌─────────────────────────────────┐
│   Todo List                      │
├─────────────────────────────────┤
│ [入力欄            ] [追加ボタン] │
├─────────────────────────────────┤
│ [全て][未完了][完了済み]          │
├─────────────────────────────────┤
│ □ Todoアイテム1        [削除]    │
│ ☑ Todoアイテム2        [削除]    │
│ □ Todoアイテム3        [削除]    │
└─────────────────────────────────┘
```

## HTML要素構成

### 必要な要素
- **ヘッダー部**: タイトル表示
- **入力エリア**: 
  - テキスト入力欄（input type="text"）
  - 追加ボタン（button）
- **フィルターエリア**:
  - 全てボタン（button）
  - 未完了ボタン（button）
  - 完了済みボタン（button）
- **Todoリスト**:
  - リストコンテナ（ul）
  - Todoアイテム（li）
    - チェックボックス（input type="checkbox"）
    - テキスト（span）
    - 削除ボタン（button）

## モック用CSS（最小限）
```css
/* 基本レイアウトのみ */
.container {
  max-width: 600px;
  margin: 0 auto;
  border: 1px solid #ccc;
  padding: 20px;
}

.todo-item {
  border: 1px solid #ddd;
  padding: 10px;
  margin: 5px 0;
}
```
